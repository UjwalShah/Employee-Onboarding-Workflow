### **Automated Workflow of Employee Onboarding Process**



## Overview

This Power Automate Workflow automatically process the Employee Onboarding Procedure



## Workflow

a)HR Submits the Microsoft form of an Employee

b)Power Automate detects the submission

c)Automatically updates the share-point list and notify to HR, Manager and IT Team about the onboarding of New Employee.

d)An Email notification is automatically sent.



## Technologies Used

a)Microsoft Forms

b)Share-Point

c)Power Automate

d)Microsoft Outlook
